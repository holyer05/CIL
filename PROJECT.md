## 项目目标

围绕无样例类增量学习（Exemplar-Free Class-Incremental Learning, EFCIL）完成一项具有清晰理论动机、可复现且不依赖历史样本存储的研究，并形成可投稿论文。

## 研究边界

- 不保存旧类训练样本作为回放数据。
- 不破坏 PyCIL baseline；新增能力必须可通过配置关闭。
- 不以堆叠 Loss、Attention 或 Adapter 作为创新本身。
- 每项实现必须对应一个可证伪的研究假设。
- 论文主张只能来自已记录、可复现的实验。

## 核心研究问题

1. 在无法访问旧样本时，如何从新任务数据中识别并补偿旧类表征漂移？
2. 哪些跨任务变化是可迁移的共享结构，哪些是不可可靠估计的类特异变化？
3. 如何在不引入复杂回放机制的前提下维护旧类决策边界？
4. 方法的收益是否来自正确机制，而不是参数量、训练时长或额外数据？

## 当前技术基线

项目基于 PyCIL，已有多种 CIL 方法、网络结构和实验配置。正式研究开始前需要完成：

- 固化 Python、PyTorch、CUDA 和依赖版本；
- 明确 CIFAR-100、ImageNet-100 等数据路径；
- 选定 EFCIL 对照方法与统一协议；
- 复现至少一个简单基线和一个强 EFCIL 基线；
- 建立结果、配置、日志与 Git commit 的对应关系。
